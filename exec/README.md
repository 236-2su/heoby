# exec 폴더

이 폴더는 프로젝트의 빌드, 배포 및 실행에 필요한 문서와 파일들을 포함합니다.

## 📁 폴더 구조

```
exec/
├── README.md                    # 본 파일
├── 빌드_및_배포.md              # GitLab 소스 클론 이후 빌드 및 배포 가이드
├── 외부_서비스_정보.md          # 프로젝트에서 사용하는 외부 서비스 정보
├── 시연_시나리오.md             # 시연 순서 및 화면 설명
└── db_dump/                     # DB 덤프 파일 (최신본)
    └── heoby_dump_YYYYMMDD.sql
```

## 📄 문서 설명

### 1. 빌드_및_배포.md
- 사용한 JVM, 웹서버, WAS 제품 등의 종류와 설정 값, 버전 정보
- 빌드 시 사용되는 환경 변수 등의 내용
- 배포 시 특이사항
- DB 접속 정보 등 프로젝트에 활용되는 주요 계정 및 프로퍼티가 정의된 파일 목록

### 2. 외부_서비스_정보.md
- 소셜 인증, 포트 클라우드, 코드 컴파일 등에 활용된 외부 서비스
- 각 서비스 활용에 필요한 정보 (API 키, 설정 방법 등)

### 3. DB 덤프 파일
- `db_dump/` 폴더에 최신 DB 덤프 파일 저장
- 파일명 형식: `heoby_dump_YYYYMMDD.sql`

### 4. 시연_시나리오.md
- 시연 순서에 따른 site 화면명, 실행명(클릭 위치 등) 상세 설명

---

**주의사항:**
- 소스코드는 이 폴더에 업로드하지 않습니다
- 민감한 정보(API 키, 비밀번호 등)는 실제 값 대신 예시 값으로 작성합니다
